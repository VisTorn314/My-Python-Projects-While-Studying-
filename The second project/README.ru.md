# ⚖️ Calculator IMT (Калькулятор ИМТ)

Простой калькулятор индекса массы тела (ИМТ) в командной строке.  
Программа запрашивает вес (в килограммах) и рост (в сантиметрах), вычисляет и выводит ИМТ.

## 📦 Установка

1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файл `Calculator_IMT.py`.
3. Запустите в терминале:
   ```bash
   python Calculator_IMT.py
   ```

🚀 Использование

    Программа запускается без логотипа.

    Введите ваш вес (в килограммах) целым числом.

    Введите ваш рост (в сантиметрах) целым числом.

    Программа вычисляет ИМТ = вес / (рост в метрах)² и выводит результат.

    После одного расчёта программа завершается (цикл не предусмотрен).

📝 Пример работы

Введите вес: 70

Введите рост: 175 22.857142857142858

(Обратите внимание: результат не округляется.)

🔧 Планы по улучшению

    Правильно переводить рост из сантиметров в метры (growth / 100).

    Добавить обработку нулевых и отрицательных значений (запрашивать повторно).
