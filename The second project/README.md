# ⚖️ Calculator IMT (BMI Calculator)

A simple command‑line Body Mass Index (BMI) calculator.  
The program asks for weight (in kilograms) and height (in centimetres), then computes and displays the BMI.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 📦 Installation

1. Make sure you have Python 3.x installed.
2. Download the file `Calculator_IMT.py`.
3. Run in the terminal:
   ```bash
   python Calculator_IMT.py
   ```

🚀 Usage

    The program starts without any logo.

    Enter your weight (in kilograms) as an integer.

    Enter your height (in centimetres) as an integer.

    The program calculates BMI = weight / (height in metres)² and prints the result.

    After one calculation the program exits (no loop is implemented).

📝 Example

Введите вес: 70

Введите рост: 175 22.857142857142858

(Note: the result is not rounded.)
