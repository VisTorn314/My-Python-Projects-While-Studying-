# 💰 Conventor Currency

Простой конвертер рублей в доллары США с интерфейсом в командной строке.  
Программа запрашивает сумму в рублях, умножает её на фиксированный курс (1 RUB = 0.0128 USD) и выводит результат.

## 📦 Установка

1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файл `Conventor_currency.py`.
3. Запустите в терминале:
    ```
        python Conventor_currency.py
    ```

🚀 Использование

    При запуске отображается ASCII‑логотип.
    Введите сумму в рублях (целое число).
    Программа покажет эквивалент в долларах.
    После каждого расчёта вас спросят:
    Продолжить(Y/N)?
        Нажмите N – программа завершится.
        Нажмите Y – программа продолжит работу.
        (Обратите внимание: это нестандартное поведение, см. раздел «Известные проблемы».)

📝 Пример работы

Введите рубли для конвертации в доллары: 1000

Рублей: 1000 в Долларах: 12.8

Продолжить(Y/N)? N

Введите рубли для конвертации в доллары: 500

Рублей: 500 в Долларах: 6.4

Продолжить(Y/N)? Y

Программа закрылась
